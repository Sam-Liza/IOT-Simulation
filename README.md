# IOT-Simulation

How to run:

    . ./install_deps.sh

This will install dependencies via `pip`. Install `pip` with

    sudo easy_install pip

Run the tests:

    python test1.py
    python test2.py
    ...
    python test5.py

Make your own test:

    nano vr_simulation.py
